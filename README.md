# Novel

Simplifies work with keyboard in iOS via set of classes and extensions.

## How To Get Started

- Copy content of `Source` folder to your project.

or

- Use `Novel` cocoapod

## Requirements

* iOS 9 and later
* Xcode 9 and later
* Swift 4
